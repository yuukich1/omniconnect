# 🌐 OmniConnect Backend

> ### 🚧 Статус проекта: В разработке (Work in Progress)
>
> На данный момент начата работа на интеграцией Telegram

Асинхронный бэкенд на Python, созданный для создания **единого общего пространства**, куда будут стекаться личные сообщения из разных платформ (**Telegram** и **Discord**).

### 🎯 Как это работает?

Вместо небезопасных юзерботов, проект использует официальных ботов в качестве шлюзов. Они собирают сообщения из ваших личных чатов в мессенджерах и бережно синхронизируют их в одном месте, полностью исключая риск бана аккаунтов.

### ⚠️ Важное ограничение

Из-за специфики работы официальных ботов (особенно в Telegram), приложение **не имеет возможности самостоятельно начать диалог** с пользователем первым. Переписка в едином хабе активируется только после того, как пользователь сам напишет вашему боту.

### 🛡️ Безопасность и Стек

**Защита токенов:** Все API-ключи подключенных ботов автоматически шифруются алгоритмом AES-256 (Fernet) при сохранении в БД.
