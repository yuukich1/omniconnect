# 🌐 OmniConnect

> ### 🚧 Статус проекта: В разработке
>
> Активная работа над интеграцией Telegram и UI-интерфейса.

**OmniConnect** — это ваш единый хаб для управления перепиской из разных мессенджеров (Telegram, Discord) в одном месте. Мы объединяем все личные чаты в один удобный интерфейс, чтобы вы ничего не пропустили.

### 🎯 Как это работает?

Вместо использования сомнительных юзерботов, мы используем официальные API мессенджеров. Вы просто подключаете своего бота, и он становится «мостом» между мессенджером и OmniConnect.

### 💡 Основной сценарий (User Flow)

1. **Начало работы:** Зарегистрируйтесь в приложении и войдите в личный кабинет.
2. **Подключение бота:** * Создайте бота в Telegram через [@BotFather](https://t.me/BotFather).
   * Вставьте токен вашего бота в разделе «Боты» в OmniConnect.
3. **Синхронизация:** Как только кто-то напишет вашему боту в Telegram, чат автоматически появится в списке диалогов OmniConnect.
4. **Общение:** Отвечайте на сообщения прямо из нашего интерфейса — пользователь получит их в Telegram так, будто вы ответили с телефона.

### ⚠️ Важное ограничение

Из-за политики безопасности Telegram, бот **не может начать диалог первым**. Чтобы чат появился в OmniConnect, пользователь должен сначала написать вашему боту любое сообщение.

### 🛡️ Безопасность

Мы заботимся о вашей приватности:

* **Шифрование:** Все API-ключи от ваших ботов шифруются по стандарту **AES-256 (Fernet)**.
* **Безопасность аккаунтов:** Мы не используем ваш личный аккаунт, поэтому риск блокировки со стороны Telegram исключен.
